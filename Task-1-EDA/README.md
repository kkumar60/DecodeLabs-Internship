# Task 1 - Exploratory Data Analysis and Feature Engineering

## Objective

The objective of this task was to understand the dataset, clean the data, identify patterns, handle missing values, detect outliers, perform feature engineering, and prepare the dataset for machine learning.

---

## Work Performed

- Loaded the dataset using Pandas.
- Explored the dataset structure.
- Checked data types.
- Identified missing values.
- Removed duplicate records.
- Performed statistical analysis.
- Detected outliers using the IQR method.
- Applied outlier capping.
- Encoded categorical variables.
- Scaled numerical features.
- Saved the cleaned dataset.

---

## Libraries Used

- Pandas

- Matplotlib
- Seaborn
- Scikit-learn
